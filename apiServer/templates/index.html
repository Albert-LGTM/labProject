API Endpoints

GET /api/

Lists all the available endpoints of the API.

GET /api/authorization/<group>/<tag>/<ip>

Authorizes a user based on group, tag, and IP.

GET /api/getInfo?query_type=<type>&query_value=<value>

Gets information based on query type and value. Parameters: query_type, query_value.

GET /api/authenticated?hostname=<hostname>

Checks if a device is authenticated based on hostname. Parameters: hostname.

GET /api/user?user=<username>

Gets information about a user. Parameters: user.

GET /api/ipaddress?ip=<ip_address>

Checks if an IP address is present. Parameters: ip.
